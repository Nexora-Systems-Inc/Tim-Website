# Meeting Notes

Meeting summaries and client conversations.

Record notes from kickoff calls, design reviews, status meetings, and ad-hoc client discussions. Link back to baseline email IDs (`E-00X`) and requirements (`R-00X`) where relevant.

## Naming Convention

```
YYYY-MM-DD-[topic].md
```

Examples:

```
2026-01-10-kickoff.md
2026-02-05-design-review.md
2026-03-20-client-check-in.md
```

## Suggested Note Structure

```markdown
# [Meeting Title]

**Date:** YYYY-MM-DD
**Attendees:** [Names and roles]
**Duration:** [e.g., 45 min]

## Agenda

- [Topic 1]
- [Topic 2]

## Discussion Summary

[Key points discussed]

## Decisions

- [Decision 1] → see baseline D-00X

## Action Items

| Owner | Action | Due |
|-------|--------|-----|
| | | |

## References

- Baseline: E-001, R-003
```
