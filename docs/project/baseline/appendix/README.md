# Appendix

Supporting reference material for the project baseline.

Store files here that are referenced from [`baseline.md`](../baseline.md) — for example:

- Screenshots and design mockups
- Client markups and annotated images
- Photographs and brand assets
- PDFs, contracts, and signed documents
- Email attachments and exported threads

## Naming Convention

Use descriptive, lowercase filenames with hyphens:

```
homepage-mockup-v1.png
signed-proposal-2026-01-15.pdf
logo-files.zip
```

Register each file in the **Appendix References** table in `baseline.md` with a unique ref ID (A-001, A-002, …).
